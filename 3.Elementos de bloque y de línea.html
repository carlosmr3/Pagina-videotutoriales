Elementos de bloque y de línea.

Una explicación de porqué algunos elementos se comportan de manera distinta.

El párrafo es un elemento de bloque y ocupa todo el ancho del navegador. El siguiente elemento va debajo.

El div es otro elemento de bloque que ocupa todo el ancho

Los enlaces como las imágenes son elementos de línea, van seguidos como las letras de una línea


Para poner una imagen debajo de otra podemos meter cada imagen (podría ser solo la primera, o solo la segunda si no hubiera nada más después) dentro de un párrafo


También se puede utilizar la etiqueta "br" para poner una debajo de otra, pero seguirían estando en el mismo párrafo.
